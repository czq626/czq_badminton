# 游戏方向

## 核心定位

这个项目是一个零依赖 H5 Canvas 小游戏，方向是复刻“火柴人打羽毛球”这类童年网页小游戏的核心体验：打开即玩、规则直观、反馈清楚、双人对战有乐趣，单人模式也能快速进入状态。

当前版本更接近“高完成度原型”，主要验证玩法结构、手感方向和基础功能闭环。后续如果要追求更高还原度，需要继续对照原始视频或原版游戏实录做逐帧校准。

## 设计原则

- 轻量：不引入构建工具和运行时依赖，直接打开 `index.html` 即可运行。
- 可传播：前端可以打包为单个 HTML；真实跨设备房间服务作为外部可选增强。
- 清晰：球、球员、网、比分、发球方、胜利条件要一眼可读。
- 手感优先：挥拍时机、跳跃高度、扣杀速度、短球落点比复杂系统更重要。
- 双端可玩：键盘双人和移动端触屏都要保留。
- 可迭代：玩法参数、AI 难度、视觉反馈和文档记录都要方便后续调整。

## 目标体验

- 30 秒内理解玩法。
- 1 分钟内完成一局 7 分制比赛。
- 玩家能感受到普通回球、扣杀、短球之间的差异。
- 长回合能自然升温，玩家能感到相持、抢网和压后场之间的节奏变化。
- AI 有明显难度差异，但休闲难度不会让新玩家挫败。
- 本地双人对战要有足够的可控性和随机趣味。
- 联机模式保持轻量，优先使用房间列表服务手动约战；服务不可用时仍能离线游玩。

## 非目标

- 暂不做账号系统、天梯、自动匹配或长期持久化房间。
- 暂不引入大型游戏引擎。
- 暂不追求复杂角色养成、装备或商业化系统。
- 暂不做像素级原版复刻，除非后续拿到更完整的原始参考素材。
